---
title: Деплой
---

Данный документ поможет вам развернуть копию репозитория на своем сервере. Приложение распространяется с открытым исходиным кодом по лицензии [GPLv3](https://www.gnu.org/licenses/gpl-3.0.ru.html)

Для того чтобы развернуть приложение на своем сервере вам понадобится:

- Docker
- Аккаунт на Heroku (опционально)
- Аккаунт на Firebase

Последний используется для того чтобы хранить данные о созданных комнатах и пользователях, которые подключены к данным комнатам.

Приложение можно развернуть на любой операционной системе, так как оно распространяется с Docker-образом. Важно только наличие Docker на системе.
